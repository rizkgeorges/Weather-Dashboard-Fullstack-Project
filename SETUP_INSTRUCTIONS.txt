# Weather Dashboard - Setup Instructions

## How to Run Locally:
1. Download/clone all project files to a folder
2. Install Live Server extension in VS Code OR use Python: `python -m http.server 8000`
3. Open index.html in your browser via the local server (NOT by double-clicking)
4. Navigate between pages using the top navigation

## Testing Checklist:
- Weather dashboard loads with city cards
- Click cards to see 7-day forecast modal
- Contact form shows success message when submitted
- FAQ accordion opens/closes
- Gallery images load and open in modal
- Navigation works on all pages
- Mobile menu toggles properly

## Requirements:
- Modern browser (Chrome, Firefox, Safari, Edge)
- Internet connection for API calls
